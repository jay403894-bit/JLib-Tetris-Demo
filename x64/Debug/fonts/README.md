# Aldrich

`Aldrich-Regular.fnt` / `Aldrich-Regular.png` is a bitmap-font atlas baked from the
[Aldrich](https://fonts.google.com/specimen/Aldrich) typeface by Matthew Desmond (MADType),
distributed as a free font via Google Fonts.
